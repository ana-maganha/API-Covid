PÁGINA INICIAL SOBRE EQUIPE

Este projeto faz a descrição de dados referente a Covid-19, incluindo um mapa mundial com casos da doença por todos os países do mundo, um gráfico que traz número e porcentagem de casos confirmados, mortos, e recuperados mundialmente, e por fim uma tabela que apresenta os totais de casos confirmados, mortes, suspeitos e descartados por Estados do Brasil.

Com isso, seu objetivo se define em apresentar os dados e alertar a população diante ao agravamento na saúde durante a pandemia e até os dias atuais, já que não foi extinta a doença.

Para este trabalho foram utilizados ferramentas e tecnologias específicas, incluindo programação em HTML, CSS e Java Script. Sendo aplicado a programação de API, com dashboard e web responsiva.
